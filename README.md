# MedAI V3 — AI-Powered Medical Assistant

A modern, production-ready SaaS medical platform designed to help doctors collect patient data, analyze symptoms, assist with differential diagnosis, and generate structured medical reports.

## Features

- **Landing Page** — Professional hero section with feature highlights and HIPAA compliance notice
- **Secure Login** — Doctor authentication portal
- **Dashboard** — Overview of patient reports, stats, and search functionality
- **Multi-Step Patient Data Collection** — Structured forms with validation, progress tracking, and JSON output
- **AI Symptom Analysis** — Simulated AI analysis with differential diagnoses, lab test suggestions, red-flag warnings, and next steps
- **Medical Report Generator** — Full structured reports with editable clinical notes, copy-to-clipboard, and print/PDF support
- **Dark Mode** — Full dark/light theme toggle
- **Responsive Design** — Works seamlessly on mobile and desktop

## Tech Stack

- React 18
- Vite
- Pure CSS-in-JS (no external UI libraries)
- DM Sans + Instrument Serif typography

## Getting Started

```bash
npm install
npm run dev
```
